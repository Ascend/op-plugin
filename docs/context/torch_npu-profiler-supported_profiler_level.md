# torch_npu.profiler.supported_profiler_level

## 函数原型

```
torch_npu.profiler.supported_profiler_level()
```

## 功能说明

查询当前支持的torch_npu.profiler.ProfilerLevel级别。

## 支持的型号

- <term>Atlas A2 训练系列产品</term>
- <term>Atlas A3 训练系列产品</term>
