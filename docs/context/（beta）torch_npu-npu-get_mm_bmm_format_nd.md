## 功能说明

确认线性module里面的mm和bmm算子是否有使能ND格式，如果使能了ND，返回True，否则，返回False。

## 函数原型

```
torch_npu.npu.get_mm_bmm_format_nd()
```

## 返回值说明

`bool`


## 调用示例

```python
>>> torch_npu.npu.get_mm_bmm_format_nd()
True
```
