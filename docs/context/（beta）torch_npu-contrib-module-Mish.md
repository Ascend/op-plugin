## 功能说明

应用基于NPU的Mish操作。

## 函数原型

```
torch_npu.contrib.module.Mish(nn.Module)
```

## 调用示例

```python
>>> from torch_npu.contrib.module import Mish
>>> m = Mish()
>>> input_tensor = torch.randn(2, 32, 5, 5).npu()
>>> output = m(input_tensor)
```
