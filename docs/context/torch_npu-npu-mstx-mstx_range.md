# torch_npu.npu.mstx.mstx_range

## 函数原型

```
torch_npu.npu.mstx.mstx_range(msg: str, stream = None, domain: str = 'default')
```

## 功能说明

range装饰器，用来采集被装饰函数的range执行耗时。
