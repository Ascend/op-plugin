# torch_npu.npu.mstx

## 函数原型

```
torch_npu.npu.mstx()
```

## 功能说明

打点接口。

用于为[torch_npu.profiler._ExperimentalConfig](torch_npu-profiler-_ExperimentalConfig.md)的msprof_tx提供打点接口调用。
