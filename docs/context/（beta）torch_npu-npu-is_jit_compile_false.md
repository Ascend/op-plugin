## 功能说明

确认算子计算是否采用二进制，如果是二进制计算，返回True，否则返回False。

## 函数原型

```
torch_npu.npu.is_jit_compile_false()
```
## 返回值说明

**bool型**。




## 调用示例

```python
>>>torch_npu.npu.set_compile_mode(jit_compile=False)
>>>torch_npu.npu.is_jit_compile_false()
True
```
