# （beta）torch_npu.contrib.module.NpuCachedDropout

## 函数原型

```
torch_npu.contrib.module.NpuCachedDropout(torch.nn.Dropout)
```

## 功能说明

在NPU设备上使用FairseqDropout。

## 参数说明

- p (Float) - 元素归零的概率。
- module_name (String) - 模型名称。
