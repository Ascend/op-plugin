## 功能说明

应用基于NPU的Sigmoid线性单元（SiLU）函数，按元素方向。SiLU函数也称为Swish函数。

## 函数原型

```
torch_npu.contrib.Swish()
```

## 调用示例

```python
>>> m = torch_npu.contrib.Swish().npu()
>>> input_tensor = torch.randn(2, 32, 5, 5).npu()
>>> output = m(input_tensor)
```
