## 功能说明

按元素计算self的双曲正切。

## 函数原型

```
torch_npu.npu_mish(self) -> Tensor
```

## 参数说明

**self**（`Tensor`）：数据类型支持`float16`、`float32`。

## 调用示例

```python
>>> x = torch.rand(10, 30, 10).npu()
>>> y = torch_npu.npu_mish(x)
>>> y.shape
torch.Size([10, 30, 10])
```
