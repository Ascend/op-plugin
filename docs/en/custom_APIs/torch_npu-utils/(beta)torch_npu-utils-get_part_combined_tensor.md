## Function

Obtains a local tensor from a fused tensor generated by `torch_npu.utils.npu_combine_tensors` based on the address offset and memory size.

## Prototype

```python
torch_npu.utils.get_part_combined_tensor(combined_tensor, index, size) -> Tensor
```

## Parameters

- **`combined_tensor`** (`Tensor`): Fused tensor generated by `torch_npu.utils.npu_combine_tensors`.
- **`index`** (`Long`): Offset address of the local tensor relative to the fused tensor.
- `size` (`Long`): Size of the local tensor.

## Return Values

`Tensor`

Local tensor obtained from the fused tensor.

## Constraints

The sum of `index` + `size` must be less than or equal to the memory size of `combined_tensor`.
