## Function

Validates whether all tensors in a tensor list belong to a single fused tensor generated by `torch_npu.utils.npu_combine_tensors`.

## Prototype

```python
torch_npu.utils.is_combined_tensor_valid(combined_tensor, list_of_tensor) -> bool
```

## Parameters

- **`combined_tensor`** (`Tensor`): Fused tensor generated by `torch_npu.utils.npu_combine_tensors`.
- **`list_of_tensor`** (`List[Tensor]`): Required. Tensor list to be validated.

## Return Values

`bool`

Indicates whether all tensors in `list_of_tensor` belong to the fused tensor `combined_tensor`.

## Constraints

The fused tensor `combined_tensor` and all tensors in `list_of_tensor` must be memory-contiguous NPU tensors. The data types of these tensors must be identical.
